Tables

Element Description
<table> Tag enclosing the whole table content
<tr> Table row. Encloses table data (cells)
<td> Table data (cell). Child of the <tr> element
<th> Table header element
<thead> Child of <table>, encloses the table head
<tbody> Child of <table>, encloses the table body

What are tables for?

Tables should - suprisingly - be used to hold tabular data. They should not be used for HTML pagestyling!
